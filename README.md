## Design requirements

- 싱글 페이지로
- 클릭을 작게 할 수 있게
- 깔끔하지만 나의 개성을 살려서
- 전문가, 전문 지식, 개발 관련
- 개발에 대한 나의 열정
- 나의 기술 스택
- 내가 가지고 있는 기술, 언어, 툴
- 프로젝트들(카테고리, 우선순위)
- 추천사
- 연락처
- 링크: Github, LinkedIn
